Require to have golang installed : 

API scripts : 
1. Create Queue : 
> ./scripts/create_queue.sh
2. register subscriber to queue
> ./scripts/register_subscriber.sh
3. publish message
> ./scripts/publish_message.sh
4. poll message
> ./scripts/poll_message.sh
5. Ack message
> ./scripts/ack_message.sh

Compile using make : 
> make <Enter>
